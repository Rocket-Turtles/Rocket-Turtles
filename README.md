## Development

### Installing Dependencies

From within the root directory:

```sh
npm install
```

### Manipulating Postgresql from Heroku

Resetting db
```sh
heroku pg:reset DATABASE
```

Migrating DB
```sh
heroku run knex migrate:latest
```

Seeding DB
```sh
heroku run knex seed:run
```

Checking logs
```sh
heroku logs --tail
```

Opening pate
```sh
heroku open
```

### Deployment

```sh
git push heroku master
```
